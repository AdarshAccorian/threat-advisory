SQL Injection Vulnerability in projectworlds Online Admission System 1.0

CVE-2025-12938 | September 3, 2025
HIGHSeverity
7.3CVSS Score

Description

This vulnerability allows an attacker to manipulate the 'keyword' argument in a PHP script, potentially leading to unauthorized database access and data modification.

Recommendations

To mitigate this vulnerability, ensure that any input arguments passed to file /process_login.php are sanitized or validated against a predefined set of allowed keywords. This can be achieved by using prepared statements or parameterized queries when making database requests from PHP.

Password Generation Algorithm Vulnerability

CVE-2025-41731 | November 10, 2025
CRITICALSeverity
9.4CVSS Score

Description

An unauthenticated local attacker with knowledge of the password generation timeframe might be able to brute force the password in a timely manner, allowing them to gain root access to the device if the debug interface is still enabled.

Recommendations

Implement a rate limiting mechanism on the password generation endpoint to limit the number of simultaneous requests, and validate user input to prevent brute-force attacks. Also, consider implementing a password blacklisting feature to block known compromised passwords.

CVSS Vector

CVSS:3.1/AV:L/AC:H/PR:N/UI:N/S:U/C:H/I:H/A:H

Matching Domain
